# 移动端访问使用指南

本指南将帮助您方便地使用手机访问实验室管理系统，实现移动办公和操作。

## 快速访问方式

### 方法一：扫描二维码访问

1. **打开移动端访问指南**
   在电脑上打开生成的访问指南文件：`mobile_access_guide_20251010_144955.html`
   
2. **扫描QR码**
   使用手机上的浏览器或专门的扫码应用，扫描页面中的二维码
   
3. **自动跳转**
   扫描成功后，手机将自动跳转到移动端登录页面

### 方法二：手动输入网址访问

如果您无法扫描二维码，可以直接在手机浏览器中输入以下网址：

- **局域网访问地址**：`http://172.30.81.97:8000/mobile_login_final.html`
- **本机访问地址**（仅适用于手机连接电脑热点的情况）：`http://localhost:8000/mobile_login_final.html`

## 注意事项

1. **网络连接要求**
   - 确保您的手机和运行系统的电脑连接在同一个局域网内
   - 如果是在不同网络环境下，需要进行网络配置（详见网络配置指南）

2. **浏览器兼容性**
   - 建议使用最新版的Chrome、Safari或Edge浏览器访问
   - 确保浏览器支持HTML5和JavaScript

3. **响应式设计**
   - 移动端页面已针对手机屏幕进行优化
   - 支持横屏和竖屏两种显示模式

4. **缓存问题**
   - 如果页面显示异常，尝试清除浏览器缓存或使用无痕模式访问

## 故障排除

如果您在访问过程中遇到问题，可以尝试以下解决方法：

1. **无法访问页面**
   - 检查手机和电脑是否在同一局域网
   - 确认Uvicorn服务器是否正在运行
   - 尝试使用IP地址而非主机名访问

2. **页面加载缓慢**
   - 检查网络连接速度
   - 关闭手机上的VPN或代理软件
   - 尝试刷新页面或重新启动浏览器

3. **功能异常**
   - 确认您的登录状态是否有效
   - 检查是否有网络请求错误（可在浏览器开发者工具中查看）
   - 尝试重新登录系统

## 常见问题解答

**Q: 我的手机和电脑不在同一网络，如何访问？**
**A:** 可以通过配置端口映射或使用内网穿透工具，如ngrok等。详细配置请参考网络配置指南。

**Q: 如何将常用页面添加到手机主屏幕？**
**A:** 在手机浏览器中打开页面后，使用"添加到主屏幕"功能（不同浏览器操作略有不同）。

**Q: 移动端支持所有功能吗？**
**A:** 移动端主要针对常用操作进行了优化，部分复杂配置和管理功能建议在电脑端使用。

如有其他问题，请联系系统管理员或查看完整的用户手册。
