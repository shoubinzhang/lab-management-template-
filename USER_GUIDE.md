## 🚀 系统启动

### 自动启动（推荐）
```bash
# 运行一键启动脚本
./start-system.bat
```

### 手动启动
```bash
# 1. 启动后端服务
cd backend
python -m uvicorn app:app --host 0.0.0.0 --port 8000 --reload

# 2. 启动前端服务（新终端）
cd frontend
npm start
```

## 🎯 主要功能

### 快捷操作面板
登录后主页显示快捷操作面板，包含：

#### 📊 快速统计
- 设备总数及状态
- 试剂库存情况
- 今日预约数量
- 待处理任务

#### ⚡ 常用操作
- **设备管理**：查看、预约、维护设备
- **试剂管理**：库存查询、申请、入库
- **实验记录**：创建、查看实验记录
- **报告生成**：导出各类统计报告

#### 🔔 通知中心
- 设备维护提醒
- 试剂到期警告
- 预约确认通知
- 系统公告

### 设备管理
1. **查看设备**：
   - 进入「设备管理」页面
   - 查看设备列表、状态、位置
   - 点击设备查看详细信息

2. **预约设备**：
   - 选择需要的设备
   - 点击「预约」按钮
   - 填写预约时间和用途
   - 提交预约申请

3. **设备维护**：
   - 管理员可标记设备状态
   - 记录维护日志
   - 设置维护提醒

### 试剂管理
1. **库存查询**：
   - 进入「试剂管理」页面
   - 搜索或筛选试剂
   - 查看库存数量和到期时间

2. **试剂申请**：
   - 点击「申请试剂」
   - 填写申请信息
   - 提交申请等待审批

3. **入库管理**：
   - 管理员可进行试剂入库
   - 更新库存信息
   - 设置到期提醒

### 实验记录
1. **创建记录**：
   - 点击「新建实验记录」
   - 填写实验信息
   - 上传相关文件
   - 保存记录

2. **查看记录**：
   - 浏览历史实验记录
   - 搜索特定实验
   - 导出实验数据

## 📱 移动端使用

### PWA安装
1. 在手机浏览器中访问系统
2. 点击浏览器菜单中的「添加到主屏幕」
3. 确认安装PWA应用
4. 从主屏幕图标直接启动

### 移动端特性
- **离线访问**：部分功能支持离线使用
- **推送通知**：重要提醒推送到手机
- **快捷操作**：底部快捷栏便于操作
- **响应式设计**：自适应手机屏幕

## 🔧 常见问题

### 无法访问系统
1. 检查网络连接
2. 确认服务器IP地址正确
3. 检查防火墙设置
4. 联系系统管理员

### 登录失败
1. 检查用户名和密码
2. 确认账号是否被锁定
3. 尝试重置密码
4. 联系管理员重置账号

### 功能异常
1. 刷新页面重试
2. 清除浏览器缓存
3. 尝试其他浏览器
4. 检查网络稳定性

### 数据同步问题
1. 检查网络连接
2. 等待自动同步完成
3. 手动刷新页面
4. 联系技术支持
